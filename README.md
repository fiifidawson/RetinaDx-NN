# RetinaDx-NN
Quantification of retinal morphology to detect neurodegeneration.

## Datasets
* [Retinal OCT Images - Kaggle ](https://www.kaggle.com/datasets/paultimothymooney/kermany2018/data)
* Paper
  * [Identifying Medical Diagnoses and Treatable Diseases by Image-Based Deep Learning](https://www.cell.com/cell/fulltext/S0092-8674(18)30154-5)

* [MESSIDOR -  Methods to Evaluate Segmentation and Indexing Techniques in the field of Retinal Ophthalmology (in French)](https://www.adcis.net/en/third-party/messidor/)
